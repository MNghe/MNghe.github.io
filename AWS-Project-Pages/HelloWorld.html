Run a Serverless "Hello, World! with AWS Lambda"
The Tutorial

THE QUEST

The objective is to learn the basics of running code on AWS Lambda without provisioning or managing servers. Through this tutorial, there will also be a moment where we test and monitor the output of the function.

OVERVIEW

As this project was a beginner step into the world of AWS, there is no need to go in-depth with the entire process. The initial step is to have an AWS account, a Free Tier is perfectly fine, in order to use the Lambda Console service. After arriving to the Lambda page, we choose to create a function, select the Blueprints option, and choose the hello-world-python blueprint. After a few more steps, we've reached the point where we configured the test event's value into the desired outcome. Once tested, an execution result, summary, and a log output is displayed that showed information regarding duraction, resources configured, billed duration, and more.
